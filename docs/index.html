[　DBSJ学生企画　第十二弾　] 株式会社ソラコム 片山さんに聞く！ 〜IoTの民主化を目指して大企業からスタートアップへ〜

10/11(水) 19:00～20:00 開催

DBSJ学生企画第十二弾では、株式会社ソラコムの片山暁雄さんをお招きし、オンライン配信イベントを行います。
片山暁雄さんは、株式会社ソラコムの創業時に株式会社Amazon Data Services Japan(AWS) から転職されており、現在は日本のエンジニアチームのトップとしてIoTプラットフォームの設計構築と、組織マネジメントを担当をされておられます。今回のイベントでは、IoTの民主化やIoTプラットフォームSORACOMなどのIoTにまつわることや、大企業からスタートアップに参加した時のお話をしていただきます。また、皆様からお寄せいただいたご質問・トークテーマを元にインタビューを行う予定です。
皆様からのご質問・トークテーマをお待ちしております！
(例：IoTプラットフォーム設計する上で大変なことって何ですか？
　　大企業からスタートアップに入社する時は不安ではありませんでしたか？...)
参加登録・ご質問はページ下部のフォームよりお願いいたします。
