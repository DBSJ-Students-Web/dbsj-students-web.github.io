[　DBSJ学生企画　第十六弾　] 進学？就職？研究？起業？ ― 多様なキャリアから学ぶ現代の就活術 ―

11/21(金) 18:30〜19:30 開催

開催形式：ハイブリッド（Zoom、一橋講堂（予定））

DBSJ学生企画による特別イベントを開催します！
学生の皆様が多様なキャリアパスの実例を知り、自身のキャリア選択の視野を広げ、キャリアに迷う学生が安心して相談できる場を提供することを目的としています。
修士進学・博士進学・就職・アカデミア・起業のリアルな体験談を共有し、学生が自分に合った進路を主体的に考えるきっかけをつくります。
さまざまなバックグラウンドを持つ4名による登壇で、1人あたり発表8分・質疑5分のミニトークセッションを行います！
